## Get an Executable for Your Computer

You can download a collection of precompiled binaries from the latest
[GitHub release](https://github.com/Joecstarr/pdgl/releases/). These binaries are built from within
a GitHub action. The Linux binaries on an Ubuntu image running the development shell defined by
`flake.nix`, and the Windows binaries on a Windows image.

If a binary doesn't run on your system you can open an issue to request support and/or attempt to
build from source.

[:fontawesome-solid-bug: Report!](https://github.com/Joecstarr/pdgl/issues/new/choose){ .md-button }
